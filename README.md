# SinemalardaNeVar
Eklediğiniz sinemalarda ki filmleri bulan, imdb puanlarına göre sıralayan python scripti.

Gereklilikler:
```
flask
requests
bs4
imdbpie
```

#Kullanım

views.py içerisindeki siteler listesini güncelle. (Sadece sinemalar.com verileri çalışır)

terminal
```
python run.py
```
